# DissertationGame

「論文用ゲーム」

（概要・テーマ）
この作品は、デジタルハリウッド大学院の、修了課題で開発した作品です。
簡易的なゲームでもテーマを伝えられるか？という論文を書くために制作しました。

(開発時期・期間)
デジタルハリウッド大学院2年生時の2021年10月～2021年12月(3ヶ月)

（開発ツール）
・ゲームエンジン：Unity(バージョン2020.3.16f1)
・言語：C#
・開発環境：Visual Studio Code
・2D データ : イラストレーター、一部フリー素材を使用

（チームメンバー）
・プログラマー1人（担当）

（ゲームシステム）
行動の選択肢を通して特定日数生き残る必要があるサバイバルゲームです。

(操作方法)
・マウスでボタンをクリックして進めてください。
・ESCキーでゲームを終了します。

(起動方法)
「Dissertation(ビルド)」というフォルダーを開いてください。
その中にある、「dissertation_Kashiwabara.exe」というアプリケーションをダブルクリックしてください。
